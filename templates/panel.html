Chat commands

Only you and your moderators can trigger these:

  • !nobot on — enable chat-triggered "no" responses (default when you go live)
  • !nobot off — disable chat triggers for the rest of this stream
  • !nobothelp — noBot whispers you the link to this panel

When on, saying a simple "no" / "nope" / "nah" / "hell no" / etc. in your own chat drops a funnier version in after you.

This button

The big red button posts a random funny "no" as noBot. It always fires — it ignores the !nobot on/off toggle and has no cooldown. Press away.

Add as an OBS dock

  1. In OBS: View → Docks → Custom Browser Docks…
  2. Name: noBot
  3. URL: paste the address bar of this page (includes ?k=…)
  4. Click Apply, then drag the dock into your layout.

For an on-stream overlay instead: Sources → + → Browser, same URL.
